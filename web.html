Front end web design

The website you are currently viewing is my attempt to showcase some of the skills I've developed over the past few years. That includes using HTML, CSS, bootstrap and a little javascript to build this site. Some of the elements you'll see are:

  • minimalist palette and styling;
  • internal and external links;
  • font sourced from googlefonts API;
  • a navigation bar which uses a bootstrap template;
  • the use of 'flexbox' for layout and responsive objects;
  • fontawesome icons;
  • bootstrap 'cards' as elements of a portfolio;
  • 'stretched links' allowing users to click on objects, areas or images;
  • the use of 'iframes' to display external HTML content;
  • a photography 'slideshow';
  • CSS styling to highlight objects on hover via shadows, transformations and z-index.
